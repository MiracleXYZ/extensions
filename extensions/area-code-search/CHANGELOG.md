# Changelog

All notable changes to this extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Initial Release] - 2026-07-16

### Added

- Search US area codes by area code digits, city name, or state name
- Results grouped by area code with all linked cities for overlay inspection
- Timezone label (ET, CT, MT, PT, etc.) and current local time per city
- Copy actions: area code only, city/state location, or combined "area code — city, state"
- Empty states for initial view and no-results

### Changed

- Extension title set to "Area Code Search" for clarity
- Dependencies updated to latest compatible versions
- Single, focused action panel for copy operations
