# Screenshots

This directory should contain screenshots for the Raycast Store listing.

## Requirements

- **Resolution**: 2000×1250 pixels (16:10 aspect ratio)
- **Format**: PNG
- **Count**: Minimum 3, maximum 6 screenshots

## Naming Convention

- `area-code-search-1.png` - Initial search view
- `area-code-search-2.png` - Search results
- `area-code-search-3.png` - Action panel showing copy location option

## Guidelines

- Use high-contrast backgrounds
- Keep backgrounds consistent across all screenshots
- Showcase the most useful features
- Avoid including sensitive data
- Use consistent theming (all light or all dark mode)

## How to Create Screenshots

1. Run the extension in development mode: `npm run dev`
2. Use Raycast's built-in screenshot tool (⌘⇧5) to capture the window
3. Resize images to 2000×1250 pixels
4. Save as PNG files in this directory with the naming convention above
