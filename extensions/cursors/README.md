# Cursors

All macOS cursors at your fingertips.

Search the full set of macOS system cursors and copy them straight into your design tools.

## Features

- **Search** every macOS cursor in a grid.
- **Copy** or **Paste** a cursor as SVG markup (the primary action is configurable in preferences).
- **Copy / Paste / Save as PNG** — export any cursor as a transparent PNG at **16, 32, 64, 128, 256, or 512px**. PNGs are rendered from the source SVG as true vector art, so they stay crisp at every size.
  - **Copy as PNG** → puts the image file on the clipboard (paste it into Figma, Sketch, Slack, etc.).
  - **Paste as PNG** → pastes the image directly into the frontmost app.
  - **Save as PNG** → writes the file to your Downloads folder and reveals it in Finder.
- **Quick Look** (⌘Y) — preview any cursor full-size without leaving Raycast.
- **Set Preview Backdrop** — put the grid on a **White**, **Black**, or **50% Gray** well (defaults to None). Handy for judging light cursors on light and dark cursors on dark — the `crosshair`, for instance, is near-invisible on the default dark grid. Backdrops affect the preview only; exported PNGs stay fully transparent.
- **Copy Cursor Name** for quick reference.

macOS-only cursors that have no CSS/HTML equivalent (like `poof` and `beachball`) are marked with a ⭐️ accessory in the grid.

## Preferences

- **Primary action** — whether the top action copies or pastes the SVG.
- **Grid Size** — number of columns (3–8).
