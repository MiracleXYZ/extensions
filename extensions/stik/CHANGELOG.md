# Stik Changelog

## [Initial Version] - 2026-07-06

- Quick Capture: create notes with title, Markdown body, and optional folder
- Search Notes: full-text search across all notes
- Browse Notes: browse notes grouped by folder, with a folder filter
- Open Recent: jump to the 10 most recently modified notes
- Per-note actions: view, open in editor, copy, move between folders, delete
- Notes stored as plain `.md` files, fully compatible with the Stik app
