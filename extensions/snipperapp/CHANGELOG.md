# SnipperApp Changelog

## [Initial Version] - 2026-07-09

- Search your local SnipperApp library with workspace, language, and favorites filters
- Copy, paste, "Copy as Markdown", and open snippets; toggle favorites inline
- Quick-capture: Create Snippet form, Save Clipboard, and Save Selection commands
- Paste Last Snippet and Recent Snippets for fast re-access
- Search the SnipperApp Hub and add community snippets to your library
- Browse trending Hub snippets
- Snippets menu-bar command for one-click copy of favorites and recents
- AI Extension: let Raycast AI search, read, and create snippets, and search the Hub
- Works with no Full Disk Access required (uses the bundled SnipperApp helper)
