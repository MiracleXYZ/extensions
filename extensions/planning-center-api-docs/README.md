# Planning Center API Docs

Quickly find the docs you need for the [Planning Center API](https://api.planningcenteronline.com/docs/overview/getting-started) — without digging through the docs site.

## Features

- **Search 255+ API endpoints** across all 11 Planning Center products: People, Services, Check-Ins, Giving, Groups, Calendar, Registrations, Publishing, Webhooks, Current, and the Platform API
- **Guides and resources too** — authentication, rate limiting, webhooks, the Add-ons documentation, integration forms, the developer community, and the GitHub issues page
- **Copy docs as Markdown** (`⌘↵`) — copies an endpoint's full reference (attributes, query parameters, includable associations, ordering, nested endpoint paths, actions, and an example object) straight to your clipboard without opening a browser. Great for pasting into notes, pull requests, or an AI chat as context.
- **Copy URL** (`⇧⌘C`) — grab a link to any docs page
- **Filter by app** with the dropdown, or just type — entries are searchable by name, app, and common aliases (e.g. "bg check" finds Background Check)

## Usage

Open the **Search API Docs** command and start typing.

- `↵` opens the docs page in your browser
- `⌘↵` copies the endpoint docs as Markdown
- `⇧⌘C` copies the page URL

## Preferences

Each Planning Center product is versioned independently. By default the extension links to the latest version of each API, but you can pin any product to an older version in the extension preferences — links and copied docs will match the version you choose.
