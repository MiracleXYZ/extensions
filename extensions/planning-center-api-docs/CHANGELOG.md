# Planning Center API Docs Changelog

## [New Docs Site & New Endpoints] - 2026-07-21

- Updated all links to Planning Center's new documentation site at api.planningcenteronline.com/docs
- Added the new Current app (Organization, Person)
- Added new People endpoints: Form Field Condition, Grade, Membership Type, Workflow Assignee Summary, Workflow Assignee Workflow Summary
- Added new Calendar endpoint: Calendar
- Added new Groups endpoint: RSVP
- Added new Services endpoints: Item Assignment, Plan Import Metadata
- Added new API versions: People 2026-06-04 and 2025-11-10, Calendar 2026-06-22
- Added the Integration Page Update form link
- Added links to the Developer Community and the GitHub Issues page
- Added a "Copy Docs as Markdown" action (⌘↵) for endpoints — copies the full endpoint reference (attributes, query parameters, includes, associations, actions, and example object) without opening the browser
- Removed the "Search API Docs on Web" command (the new docs site no longer has a search results page)
