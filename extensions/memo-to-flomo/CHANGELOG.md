# MEMO to flomo Changelog

## [Initial Version] - 2026-07-03

- Added the `Send Memo` Raycast command for posting Markdown memos to flomo.
- Added required editable `MEMO API URL` extension preference.
- Added whitespace-separated multi-tag input with automatic `#tag` normalization and deduplication.
- Added local tag history with recently used tags shown in `Quick Tags`.
- Added Markdown editing support and automatic continuation for unordered and ordered lists.
