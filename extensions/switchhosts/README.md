## Required

This extension requires [SwitchHosts](https://switchhosts.vercel.app/) and its HTTP API.

Checklist:

1. Install SwitchHosts
2. Open SwitchHosts
3. Enable "Enable HTTP API" in SwitchHosts preferences

## Features

- List Hosts
- Toggle Hosts
