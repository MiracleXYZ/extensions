# BC Assessment

Search BC Assessment properties by address from Raycast.

## Features

- Search BC Assessment address matches as you type.
- Open the selected property in your default browser.
- Copy the matched address or property value.

## Usage

Open Raycast and search for `Search Address`, or type a keyword such as `bcass` followed by an address.

Example:

```text
bcass 3026 marine dr west vancouver
```
