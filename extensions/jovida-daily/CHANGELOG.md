# Jovida Daily Changelog

## [Initial Version] - 2026-07-23

- Browse todos grouped by time (Overdue / Today / Tomorrow / Later / Anytime), with search and a Pending/All filter
- Add todos with an optional AI assist that parses a plain-language note into title, due date, priority, category, subtasks, reminders, and phone-call reminder requests
- Complete (with inline Undo), reopen, edit, and delete todos — including safe handling for repeating series
- Reminders and recurring routines: create one-off or repeating todos, with notification and voice-call reminder channels surfaced in the UI
- Menu bar command showing upcoming todos grouped by time; click to complete, ⌥-click to edit
- AI Extension tools so Raycast AI can get, view, create, update, complete, delete, and precisely manage subtasks in natural language
- Device-flow sign-in; no CLI or other software to install
