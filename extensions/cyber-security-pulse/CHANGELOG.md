# Cyber Security Pulse Changelog

## [Initial Release] - 2026-07-03

- Curated cyber security RSS feeds (BleepingComputer, The Hacker News, Krebs,
  CISA, Schneier, SANS ISC) — editable via the Feed Sources preference.
- Severity classification into Critical / High / Medium / Low; red gated to
  active-exploitation or unauthenticated-RCE signals. Extra per-tier keywords are
  configurable via preferences.
- Highest non-empty tier shown inline; lower tiers as drill-downs.
- Sort dropdown: order each tier by newest-first or by criticality.
- Priority watchlist (names, aliases, or PURLs) pinned in a ⭐ Watched section.
- Denylist to hide unwanted topics.
- CISA KEV titles enriched with the affected product and CVE.
- In-Raycast preview, open in browser, copy list as markdown; ⌘→ / ⌘← navigation.
