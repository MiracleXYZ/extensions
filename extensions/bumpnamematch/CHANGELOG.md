# Bump Name Match Changelog

## [Initial Version] - 2026-07-23

- Search Names — search baby names by name or meaning with website-parity scoring, and view full details (meaning, origin, popularity, nicknames, related names, songs).
- Search by Origin — browse names by cultural origin with a gender filter.
- My Lists — view favorite lists and saved names, add names to lists, and remove them (requires a personal API key).
- Naming Sessions — vote on names one at a time with your partner, get a match toast when you both like a name, view your matches, and join a session by invite code (requires a personal API key).
