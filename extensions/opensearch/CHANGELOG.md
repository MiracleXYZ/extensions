# OpenSearch DevTools Changelog

## [Initial Version] - 2026-07-23

- Console for OpenSearch Dashboards Dev Tools syntax (`GET posts/_search` + body)
- API Explorer for arbitrary requests with a JSON viewer
- Command Palette with common OpenSearch commands
- Search command with index picker
- Query History with favorites (last 100)
- Manage Connections with multiple profiles, Basic Auth and AWS SigV4, plus a Test Connection action
- AI tool to query the cluster from Raycast AI (with confirmation for PUT/DELETE)
- Copy responses as raw JSON, pretty JSON, curl, or Dev Tools console format
